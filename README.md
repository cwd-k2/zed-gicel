# zed-gicel

[Zed](https://zed.dev) extension for [GICEL](https://github.com/cwd-k2/gicel).

## Features

- Syntax highlighting (tree-sitter)
- Code folding
- Indentation
- Bracket matching and auto-close

## Installation

### From source (dev)

```sh
# In Zed: Extensions > Install Dev Extension > select this directory
```

### Published

Once published to the Zed extension registry, search for "GICEL" in Extensions.
